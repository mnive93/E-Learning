This is an online E-learning portal which helps students study online as the materials are provided to  the user
If they have any doubts regarding the course work the students can send the queries to their faculty which will be forwarded to the faculty.
The faculty will then reply to the students which will be displayed in the students dashboard
They can also customize their dashboard with many themes available.
When ever you register the application sends a confirmation mail to your mail id 
This project has been developed in Visual studio 2010 which uses ASP.NET for front end and Microsoft SQL server for backend.
